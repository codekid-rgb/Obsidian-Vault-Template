## Then output ONLY:

**Top 3:**

2. [task]

**Also:**

## Rules
- Total output under 500 words
- Flag overload if Top 3 + Other Tasks exceeds 8 items total
- Minimal commentary unless asked
- Omit Constraints line if none given
- Save morning section to Daily/2026/YYYY-MM-DD.md
