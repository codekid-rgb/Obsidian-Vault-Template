# Weekly Review

Read ALL daily notes from this week (Monday–Sunday). Read Now/ folder for project status. Read Schedule/Upcoming Deadlines and Events.md.

## Pattern detection — look for:
- Repeated missed tasks (same task appearing multiple days)
- Overplanning vs execution gaps
- Avoidance behavior (same task deferred 3+ times)
- Energy vs productivity correlation
- Friction patterns (same type recurring)

## Reality gap check:
Total tasks planned this week vs completed. Which categories had worst completion (farm/build/school/people/faith)?
