# Daily Note Creation

Check if Daily/2026/YYYY-MM-DD.md exists for today (use actual current date).

If yes: open it and report what's already there.

If no: create it using the content of Templates/Daily Note.md, replacing {{title}} with today's date in YYYY-MM-DD format. Save to Daily/2026/YYYY-MM-DD.md.

Output one line only: "Created Daily/2026/[date].md — run /morning to begin"
